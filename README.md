# Blog

- Posts
- Categories
- Tags
- Registration and Authentication

## Requirements (Prerequisites)

* Python 3.9 and up

## Installation 

1. Create virtual environment:

```
python -m venv venv
```

2. Activate virtual environment
```
.\venv\Scripts\activate
```

3. Install requirements

```
python -m pip install -r requirements.txt
```

## Build with
* Django 4
* Bootstrap 5
* Postgres 14
